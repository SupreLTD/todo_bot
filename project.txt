Телеграм бот для управления списком задач

Описание функций:

1. Регистрация пользователя:

   - Бот должен предложить пользователю зарегистрироваться, с помощью команды /start или другой команды активации.

   - При регистрации пользователь должен указать свое имя и номер телефона

2. Добавление задачи:

   - Пользователь должен иметь возможность добавить новую задачу в свой список.

3. Просмотр списка задач:

   - Пользователь должен иметь возможность просмотреть свой список задач.

4. Управление задачей:

   - Пользователь должен иметь возможность отметить задачу как выполненную.

   - Бот должен уметь помечать задачу как выполненную и удалять ее из списка задач.

6. Поиск задач:

   - Пользователь должен иметь возможность искать задачи по ключевым словам или фразам.

   - Бот должен предоставлять пользователю результаты поиска по заданным критериям.
